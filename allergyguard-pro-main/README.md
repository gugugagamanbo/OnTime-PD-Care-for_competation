# 帕金森照护助手

面向帕金森患者和家人共同账号使用的前端原型，包含用药提醒、药物库存、扫码录入、照护圈、Apple Watch 数据摘要、就诊信息生成、账号安全和关于与免责声明等模块。

## 本地运行

```bash
npm run dev -- --host 127.0.0.1 --port 8080
```

## 开源与数据说明

本仓库不包含真实患者信息、真实医生联系方式、真实医院数据或生产环境密钥。代码中的联系人、手机号、设备和临床记录均为 Demo 示例数据。请不要将真实病历、处方图片、患者姓名、电话号码、医院内部信息或 API key 提交到仓库。

本项目是前端原型，不构成医疗器械、诊断工具或治疗建议系统。所有用药调整和临床判断应以医生或药剂师的正式建议为准。

## 主要页面

- 用药：下一次服药、今日用药时间轴、药物库存、医生/药剂师联系人弹窗。
- 照护圈：快速记录症状、照护团队成员新增/编辑/删除、AI 摘要、照护者状态。
- 扫码录入：处方或药盒识别后的待确认用药计划，以及手动添加药物。
- 我的：当前药物清单、近期报告、生成就诊信息、账号安全、隐私与授权、关于与免责声明。
